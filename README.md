# emprestimo-system

Projeto organizado para crescimento futuro, com frontend separado, base de backend, scripts de banco e documentação.

## Como publicar a página

Publique o conteúdo da pasta `frontend/` na sua hospedagem estática.

## Antes de publicar

- Ajuste `frontend/assets/js/appConfig.js` com seu número de WhatsApp
- Ajuste `frontend/assets/js/appConfig.js` com a URL do Google Apps Script
- Coloque imagens e logo reais em `frontend/assets/img/` quando quiser trocar os arquivos atuais

## Estrutura

- `frontend/`: página pública e assets
- `backend/`: base para futura API
- `database/`: scripts de banco
- `docs/`: apoio técnico
